TGA Export Options — a Krita plugin that automatically shows an options dialog every time you save or export a .tga file. Features: • Origin: Bottom-Left (standard) or Top-Left (OpenGL / game engines) • Bit Depth: keep, force 32-bit RGBA, or strip alpha to 24-bit RGB • RLE Compression: toggle on/off • TGA 2.0 Footer (TRUEVISION-XFILE signature) • Comment / Image ID (up to 255 ASCII chars) • All settings are remembered between Krita sessions The plugin hooks into Krita's imageSaved signal — no menus to navigate. Just save a .tga file and the dialog pops up automatically. Installation: copy 3 files into your pykrita/ folder, enable in Plugin Manager, restart Krita.
